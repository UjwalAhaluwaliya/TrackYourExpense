A Spring Boot application for tracking personal expenses with both traditional and OAuth2 authentication.

## Features
- 📊 Expense tracking with date, category, and amount
- 🔐 Secure authentication (form-based + OAuth2 via Google/GitHub)
- 📈 Dashboard view of all expenses
- ➕ Add/Edit/Delete expense functionality
- 👤 User registration and profile management
